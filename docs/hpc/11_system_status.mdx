# Torch System Status
:::note
To be able to see the panels below you need to be within NYU network (use VPN if you are not on campus)
:::

On the NYU VPN, you can head to [https://stats.apps.cloud.rt.nyu.edu](https://stats.apps.cloud.rt.nyu.edu/) and get a view of the current status of the cluster including but not limited to:
-   CPU Allocation
-   Memory Allocation
-   GPU Allocation (for each GPU type)
-   CPU Job Status
-   GPU Job Status
